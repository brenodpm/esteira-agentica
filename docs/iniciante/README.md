# Documentação para Iniciantes

Este diretório contém documentação voltada para usuários que estão começando com o projeto.

## Estrutura esperada

- `instalacao.md` - Guia de instalação e primeiros passos
- `tutoriais/` - Tutoriais básicos passo a passo
- `faq.md` - Perguntas frequentes de iniciantes

## Convenções

Arquivos devem incluir metadados no frontmatter para auto-geração da página inicial.
