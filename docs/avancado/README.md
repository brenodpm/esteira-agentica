# Documentação Avançada

Este diretório contém documentação para usuários experientes que precisam de configurações avançadas e conceitos técnicos detalhados.

## Estrutura esperada

- `configuracao-avancada.md` - Configurações customizadas
- `arquitetura/` - Documentos sobre arquitetura do sistema
- `integracao/` - Guias de integração com outros sistemas

## Convenções

Arquivos devem incluir metadados no frontmatter para auto-geração da página inicial.
