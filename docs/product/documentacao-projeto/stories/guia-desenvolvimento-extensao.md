# US — Guia de Desenvolvimento e Extensão

Status: draft
Owner: product
Last updated: 2026-06-12

## Inputs
- Código fonte do projeto
- Estrutura de agentes existentes
- Épicos da documentação

## Descrição
Como desenvolvedor que quer contribuir ou estender a esteira
Quero documentação sobre a arquitetura e como criar agentes personalizados
Para implementar funcionalidades específicas para meu contexto

## Regras de negócio
- Deve explicar arquitetura geral do sistema
- Deve documentar como criar agentes personalizados
- Deve incluir guia de contribuição para o projeto
- Deve explicar APIs e interfaces principais

## Critérios de aceitação
- Dado que quero criar um agente personalizado, quando sigo o guia, então consigo implementar e integrar o agente
- Dado que quero contribuir com o projeto, quando leio o guia de contribuição, então sei o processo e padrões a seguir
- Dado que preciso entender o sistema, quando leio sobre arquitetura, então compreendo fluxos e componentes principais
- Dado que quero debuggar um problema, quando consulto documentação técnica, então encontro informações sobre logs e debugging

## Não objetivos
- Documentação de casos de uso específicos de negócio
- Tutoriais de tecnologias utilizadas (Python, GitHub API)
- Configuração de ambientes de desenvolvimento específicos
