# Problem Space

## Utilidade

Descreve o contexto atual, os problemas identificados, seu impacto e a oportunidade de resolução. Complementa a vision com profundidade sobre o cenário.

## Layout de Documentação

```markdown
# Problem Space — <nome do épico>

Status: draft | approved | deprecated
Owner: product
Last updated: YYYY-MM-DD

## Inputs
- <arquivos utilizados>

## Contexto
<situação atual>

## Problemas
- ...

## Impacto
<efeitos dos problemas>

## Oportunidade
<por que resolver agora>
```

## Caminho do Arquivo

`doc/product/<slug-epic>/problem-space.md`
