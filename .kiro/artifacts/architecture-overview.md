# Architecture Overview

## Utilidade

Define a visão geral da estrutura do sistema: estilo arquitetural, componentes, responsabilidades e fluxo principal. É a referência para todas as decisões de implementação.

## Layout de Documentação

```markdown

Status: draft | approved | deprecated
Owner: architecture
Last updated: YYYY-MM-DD

## Inputs

## Visão geral
<descrição objetiva do sistema>

## Estilo arquitetural
<estilo + justificativa>

## Componentes
| Componente | Responsabilidade |
|-----------|------------------|
| ... | ... |

## Fluxo principal
<fluxo entre componentes>
```

## Caminho do Arquivo

`doc/architecture/<slug-story>/overview.md`
