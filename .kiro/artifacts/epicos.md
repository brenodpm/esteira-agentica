# Épicos

## Utilidade

Define os grandes blocos de entrega de uma demanda com escopo e limites claros. Serve como referência para derivar user stories.

## Layout de Documentação

```markdown
# Épicos — <nome da demanda>

Status: draft | approved | deprecated
Owner: product
Last updated: YYYY-MM-DD

## Inputs
- <arquivos utilizados>

## Épico: <nome>

**Objetivo:** <valor entregue>
**Escopo:** <o que inclui>
**Fora de escopo:** <limites claros>
```

## Caminho do Arquivo

`doc/product/<slug-epic>/epicos.md`
